Sobre la asignación

La Guía de Laboratorio Nº 1 de Desarrollo Web I, titulada Introducción y conceptos básicos de HTML y CSS, está pensada para completarse en aproximadamente 2 horas y tiene como propósito que nosotros, como estudiantes, demos nuestros primeros pasos en el desarrollo web. A través de esta práctica aprenderemos a reconocer la estructura básica de un documento HTML, a aplicar estilos con CSS tanto de forma embebida como mediante hojas de estilo externas, y a utilizar los selectores y propiedades esenciales para dar formato a fuentes y textos, de manera que podamos comenzar a construir páginas web con organización y un diseño atractivo.
