Desarrollo Web , la asignatura. Este curso introductorio de desarrollo web presenta los fundamentos de HTML, CSS y JavaScript , las tecnologías centrales para construir aplicaciones web

A través del aprendizaje de la estructura con HTML y el diseño con CSS, se aprende a crear sitios atractivos y funcionales; con JavaScript se adquieren las nociones básicas para añadir interactividad y contenido dinámico al front-end. Además, el curso aborda la arquitectura de las aplicaciones front-end y el papel que cada tecnología desempeña en la creación de soluciones modulares, incorporando buenas prácticas de diseño. También, se analizan prácticas nocivas en el desarrollo de software, para evitar errores comunes.

Guías de la asignatura

Guía 1. Introduce los fundamentos para crear páginas web

Describe la estructura básica de un documento HTML, muestra cómo aplicar estilos mediante estilos embebidos y hojas de estilo externas y explica los selectores CSS básicos junto con propiedades de tipografía y texto.

Guía 2. Selectores complejos y pseudo-clases

Profundiza en el uso y combinación de selectores complejos y en la utilidad de pseudo-clases y pseudo-elementos para estilos dinámicos. Además, ayuda a practicar el Modelo de Caja y cómo controlar el espacio y el flujo visual mediante ejemplos prácticos.

Guía 3. posicionamiento y visualización

Enseña los distintos tipos de posicionamiento en CSS, las propiedades de visualización, control de flujo de elementos y técnicas para maquetar diseños fijos y fluidos.

Guía 4. Presenta los principios del diseño adaptable

Ayuda a crear y estilizar formularios con los controles de HTML5, usar correctamente la meta-etiqueta viewport y definir media queries para aplicar estilos según el ancho del navegador.

FODA sobre el desempeño

Fortalezas

Domino la estructura y semántica de HTML, lo que me permite organizar el contenido de las páginas. También, estoy mejorando mucho en el uso de selectores y propiedades CSS para lograr diseños responsivos. Esto se ha debido gracias a la práctica continua a tráves de las guías y la primera mitad del proyecto que realizamos.

Oportuniades

Puedo aprovechar lo aprendido para construir un portafolio con proyectos funcionales que demuestren mis habilidades en maquetación y diseño responsivo, lo cual mejora mis posibilidades labo, como emprendimientos.

Debilidades

Necesito reforzar mis habiliades de programación en JavaScript de nivel intermedio a avanzado para poder añadir interactividad más sólida a mis páginas. Además, debo mejorar también en pruebas y control de calidad para detectar errores antes del despliegues.

Amenazas

Mi desempeño se puede ver afectado si no practico de forma constante, haciendo, desarrollo web. Además, si no organizo mi tiempo, mi frecuencia de desarrollo practico puede decrecer.

Agradecimientos

Gracias por el curso. Considero que el orden, la dificultad y la enseñanza me han ayudado a desarrollar mis habilidades de desarrollo web.
